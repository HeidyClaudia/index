Modelos Discretos de Probabilidad

Este repositorio contiene la resolución detallada de ejercicios estadísticos utilizando modelos de probabilidad discretos. El proyecto está desarrollado en R y presentado mediante Quarto.

Estructura del Proyecto

El proyecto sigue una estructura organizada para facilitar la reproducibilidad:

/Codigos: Contiene los scripts de R con librerías (librerias.R) y funciones personalizadas (funciones_auxiliares.R).

/Datos: Archivos multimedia y bases de datos utilizadas.

/Exportar: Tablas de resultados generadas durante el análisis.

/Graficas: Exportación de visualizaciones estadísticas.

index.qmd: Documento principal que integra el análisis y el código.

 Requisitos

Para ejecutar este proyecto localmente, necesitas tener instalado:

R

RStudio

Quarto

Instrucciones

Para ver el informe, abre el archivo index.qmd en RStudio y presiona el botón Render.
